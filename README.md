
# Thence Task  
A reactjs webapp.

<br/>

## Tech Stack  
**Client:** React, TypeScript, Sass.

<br/>

## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/praveen-crypto/thence_webapp.git
~~~

Go to the project directory  

~~~bash  
  cd thence_webapp
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run dev
~~~
